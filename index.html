Browser Source Timer for OBS

This is a customizable countdown/countup timer designed for use as a browser source in OBS Studio.

Usage

Add the timer URL as a browser source in OBS with the following parameters:

Base URL: https://yourusername.github.io/browser-source-timer/timer.html

Parameters:

time - Duration in seconds (e.g., 7500 for 2h 5m)
colour - Text color in hex without # (e.g., 414141)
size - Font size in pixels (e.g., 248)
background - Background color in hex without # (e.g., transparent)
radius - Border radius in pixels (e.g., 0)
font - Google Font name with + for spaces (e.g., Anton)
direction - CountDown or CountUp
loop - true or false

Example URLs:

Countdown Timer:
timer.html?time=7500&colour=414141&size=248&background=transparent&radius=0&font=Anton&direction=CountDown&loop=false
Countup Timer:
timer.html?time=3600&colour=ff0000&size=200&background=000000&radius=10&font=Roboto&direction=CountUp&loop=true
